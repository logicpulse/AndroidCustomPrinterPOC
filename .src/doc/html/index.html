Document Overview
This document details about CUSTOM android API for printers. And also explains how to use this API for android application development for CUSTOM printers.

How to use the library
Look at the PDF manual avaiable HERE.

About CUSTOM Android API
This API provides functions to access the CUSTOM printers via Bluetooth (using serial port profile - SPP), Ethernet, USB and Serial Port. It includes below functions to access the printer

  • Get the printer class for printer
  • Connect to printer
  • Close connection
  • Set timeout for read/write
  • Write / Read bytes to/from printer
  • Print text with various text attributes
  • Print image with image scaling options
  • Print Barcode and Barcode 2D
  • Cut, eject / retract and present the ticket
  • Get picture, card and barcode from scanner
